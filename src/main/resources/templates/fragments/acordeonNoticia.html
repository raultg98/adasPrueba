Podemos dar estilos a la descripción de nuestras noticias utilizando diferentes sentencias. Es importante que estas se escriban tal cual estan definidas con espacios entre ellas. Se pueden poner todas las sentencias que deseemos y estas se pueden intercalar entre sí.
  • SALTOS DE LINEA: Si queremos introducir saltos de linea tenemos que poner $saltoLinea justo antes de donde deseamos que produzca.
  • TEXTO EN NEGRITA: Si queremos introducir texto en negrita, tenemos que encapsular las palabras que tenemos en negrita de la siguiente forma:
    $negrita texto $finNegrita
  • TEXTO EN CURSIVA: Si queremos introducir texto en cursiva, al igual que en el texto en negrita tenemos que encapsular las palabras pero de esta forma:
    $cursiva texto $finCursiva
  • TEXTO SUBRAYADO: Si queremos introducir texto el cual este subrayado, es igual que en los dos casos anterior, tenemos que encapsular las palabras, pero de esta forma:
    $subrayado texto $finSubrayado.
Ejemplo de como se implementan las sentecias y de como se intercalan estas entre sí.
Como se escribiria la noticia:
Esto es un $negrita texto de $cursiva ejemplo de $finCursiva como se dan estilos a las noticias. $finNegrita $saltoLinea Vamos a $cursiva $subrayado introducirlos $finSubrayado intercalados $finCursiva para ver $saltoLinea como funcionan estos.

Esto es un texto de ejemplo de como se dan estilos a las noticias.
